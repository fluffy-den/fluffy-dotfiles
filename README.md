# Dotfiles

Personal configuration files for Arch Linux setup.

## Contents

- **bash**: Shell configuration
- **kitty**: Terminal emulator with Ayu Dark theme
- **nvim**: Neovim configuration
- **i3**: Window manager
- **polybar**: Status bar
- **rofi**: Application launcher
- **picom**: Compositor

## Installation

Run `./install.sh` to create symlinks to home directory.

## System Info

- OS: Arch Linux
- Terminal: kitty
- Editor: nvim
- Shell: bash
